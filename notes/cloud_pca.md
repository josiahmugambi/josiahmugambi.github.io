## Recertifying as a Google Cloud Professional Cloud Architect (v2) - Dec 2021

I noticed that Accredible provides credential issuers like Google Cloud the option to add their credentials to a Blockchain (currently the Bitcoin Blockchain) record for added security.

I'm thankful that I was able to successfully re-certify as a Google Cloud Professional Cloud Architect. It was NOT easy, with lots of new content, especially hybrid and multicloud after the May 2021 exam refresh. I learnt quite a bit while preparing.

If you are looking for professional support and/or material to (re)certify on Google Cloud, these might be useful:

- The official [certification guide](https://cloud.google.com/certification)
- [Cloud Skills Boost (Powered by Qwiklabs)](https://www.cloudskillsboost.google/)
- [Tech Equity Training](https://www.techequity.training/). I'm part of the network there.
